# About Frontend

Nuxt 4 app based on `portfolio` template
- [portfolio | github](https://github.com/nuxt-ui-templates/portfolio)

Features
- Nuxt 4
- Nuxt UI (which is built on Tailwindcss)
- Typescript
- Markdown (via content module)

Modules
- @nuxt/eslint
- @nuxt/image
- @nuxt/ui
- @nuxt/content
- @vueuse/nuxt
- nuxt-og-image
- motion-v/nuxt
- @nuxt/scripts
- @nuxt/test-utils
