# Readme (frontend)

## Setup

cd into the `nuxt_app` directory & install the dependencies:
```
pnpm install
```


## Development Server

Start the development server on http://localhost:3000:
```
pnpm dev
```

or...
```
npm run dev -- -o
```

## Production

Build the application for production:
```
pnpm build
```

Locally preview production build:
```
pnpm preview
```
